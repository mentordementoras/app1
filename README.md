# Gemini Canvas para Info-Productores 🎨
Herramienta diseñada para clases en vivo que permite dibujar y obtener feedback de IA en tiempo real.

## Instalación
1. Clona este repo: `git clone https://github.com/tu-usuario/nombre-repo.git`
2. Abre `index.html` en tu navegador.
3. (Opcional) Configura tu API Key de Gemini en un archivo `.env`.

## Características
- Lienzo infinito.
- Integración con Gemini 1.5 Flash para análisis de diagramas.
- Optimizado para streaming.
